Built With Flutter

By using the Flutter mobile framework, the app is cross-platform compatible for both iOS and Android devices

Material design language inspired by Google keeps the UI clean, clutter free, and consistent

The app is written in Dart, and then compiled to native Android and iOS

Firebase Integration

User login and information is handled via Firebase provided by Google

Users register through their email and are required to perform two-factor authentication for added security

User data is securely managed, and is not hosted by the app itself

Skil Tracking

Army medical personell can log learned skills as they progress

Skills can be requested for verification by a supervisor

Supervisors have the ability to update skills in real-time

Skill Checklist

Each skill is broken down into component tasks

Tasks are specified according to the US Army documentation

Skills can be marked as in progress, when not every task is completed

Cloud Firestore

MOS, Skills, and Tasks are managed via Cloud Firestore

Custom data structures mimic the layout of data in the Army DTMS database

Supervisors can manage the catalog from within the app, and do not need to access the Firebase Console
