Unit Conversion

Users can perform calculations between units within the same family such as: ft, cm, in, m

Units other than the ones included can be added to the default list easily through the user interface

Conversion ratios for non-default units can also be specified using the like-units tool within the file menu

Full Color Theming

The color scheme of the interface can be completely changed using the built-in editor panel

The widgets, text color, logo picture, and background of the interface are all changed individually

The text color will automatically change with the widgets to promote readability

Default Themes

There are included dark theme and light themes for quick switching of interface color settings

All components will be set to the dark or light preset upon the clicking of the button

The text color will automatically change with the widgets to promote readability
