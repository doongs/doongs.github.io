Unit Conversion

United is a calculator designed to allow users to perform calculations regardless of unit differences. The app handles all conversions between units of the same relative family.

Users can add to a list of units, and can even specify their own conversion rates between units that are not from relative families. United makes calculations between Metric, Si, and US Customary units stress-free via its automatic conversion.

Full Color Theming

The color scheme of the interface can be completely changed using the built-in editor panel. The widgets, text color, logo picture, and background of the interface are all changed individually and can be specified via the interface sliders or input of RGB values ranging from 0 to 255. The text color of the widgets will automatically change with the widgets to promote readability, even for those who suffer from colorblindness.

United was built using the Java Swing GUI framework, which ensures a consistent look and feel across operating systems. United functions identically on Mac, PC, and Linux operating systems.

Default Themes

There are included dark theme and light themes for quick switching of interface color settings. All comp onents will be set to the dark or light preset upon the clicking of the button.

As with color theming, the text color will automatically change with the widgets to promote readability.
